## 🎨 四、产品功能详解

### 4.1 数据导入中心

#### 功能列表

| 功能 | 描述 | 优先级 | 工作量 |
|------|------|--------|--------|
| Excel 导入 | 支持 xls/xlsx，批量导入 | P0 | 2 天 |
| 数据去重 | 手机号去重，合并重复数据 | P0 | 1 天 |
| 数据验证 | 手机号格式验证，必填项检查 | P0 | 1 天 |
| 导入预览 | 导入前预览，确认数据 | P0 | 1 天 |
| 导入历史 | 查看导入记录，支持回滚 | P1 | 2 天 |
| API 同步 | 对接外部系统，自动同步 | P1 | 3 天 |

#### 用户流程

```
1. 上传 Excel 文件
   ↓
2. 字段映射 (自动识别)
   ↓
3. 数据预览 (显示前 10 条)
   ↓
4. 确认导入
   ↓
5. 导入结果 (成功/失败统计)
   ↓
6. 下载报告 (失败记录)
```

#### 意向打分模型

```
维度           权重    评分项
─────────────────────────────────
响应度 (30%)    30%    回复速度、回复长度、主动性
兴趣度 (35%)    35%    产品关键词、价格询问、功能细节
紧迫度 (20%)    20%    时间表达、决策周期
匹配度 (15%)    15%    用户画像匹配程度

等级划分:
  A 类 (80-100 分): 高意向 → 立即人工跟进
  B 类 (60-79 分):  中意向 → 持续培育
  C 类 (40-59 分):  低意向 → 自动化触达
  D 类 (0-39 分):   无意向 → 归档
```
